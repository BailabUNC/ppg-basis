# ppg-basis
A python-based package to automate the decomposition photoplethysmographs into various basis functions for subsequent analysis and PPG reconstruction.

# profiling
Run profile_full_pipeline to profile the three pipelines, and open the .prof files.
Run view_profile.py to open all generated .prof files
*_pipeline files refer to non-profiling pipelines
testbench_* files refer to profiling pipelines
